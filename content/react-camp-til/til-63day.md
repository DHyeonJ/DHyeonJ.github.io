# 기술면접(상태관리를 하는 이유)

## 상태관리란?

- 서비스의 규모가 커질수록 관리해야할 데이터가 많아지고 이를 체계적으로 관리하기 위해 상태 관리라는 라이브러리를 사용한다.

- 부모 컴포넌트에서 props로 자식 컴포넌트로 데이터가 전해지게 되는데 깊이가 길어지면 어디서 전해지는 데이터인지 알 수 없기 때문에 `Props Drilling` 현상을 막기 위해 상태관리를 사용한다.
